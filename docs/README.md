## 🔥 工具介绍

**枫叶指令宏**——是一款轻量级的Windows自动化脚本引擎，基于AutoHotkey v2编写，采用极简的纯文本指令语法，实现对鼠标、键盘、窗口、颜色查找、图像识别、文字识别（OCR）等功能的精确控制。它内置了、全局变量、循环、判断、模块、等，让自动化任务编写像写清单一样简单直观。

## 🛠️ 核心功能
1. **基础操作模块**：支持随机/固定延迟、全鼠标操作（单击/长按/滚轮/绝对/相对/识图移动）、键盘按键/长按模拟，搭配变量定义、数值运算、随机数、字符提取，实现拟人化基础操作
2. **图像识别模块**：支持区域识图（含识别一次/等待出现/等待消失模式）、抓图存数据、全屏/区域截图，返回匹配坐标、数量与标签，精准定位屏幕图像
3. **颜色识别模块**：支持单点坐标找色、矩形区域找色（带颜色渐变容差），自动返回匹配结果与坐标，用于界面状态判断
4. **流程控制模块**：支持单/多条件判断（与/或）、固定/无限循环、模块定义/同步调用/定时异步调用，可编写复杂自动化逻辑
5. **文本识别模块**：支持模拟文本输入、系统语音播报、字库OCR/大模型OCR识别，支持字符检索、坐标提取、字符串处理
6. **窗口系统模块**：支持窗口激活/移动/改大小/后台操作、程序运行/关闭、系统关机，可获取鼠标/窗口/系统时间状态
7. **辅助提示模块**：支持弹窗提示、悬浮提示、坐标标记，实时展示脚本运行状态，便于调试与监控

---

替代传统按键宏、取色宏工具，适配办公自动化、合法游戏辅助、软件测试、系统运维等多场景。

本工具主打「简单易用、灵活定制、拟人化操作」，支持脚本模块化复用、窗口句柄识别、操作日志记录，降低自动化门槛，提升重复操作效率。
